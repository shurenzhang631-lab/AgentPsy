# 故障排除

## 常见问题

### 测评任务长时间处于pending状态
可能原因:
1. AI模型服务不可用
2. 网络连接问题
3. 系统负载过高

解决方案:
1. 检查AI服务状态
2. 验证网络连接
3. 等待系统负载降低或联系技术支持

### 测评结果异常
可能原因:
1. 模型输出不符合预期格式
2. 网络传输数据损坏
3. 算法处理错误

解决方案:
1. 重新运行测评任务
2. 检查模型配置
3. 联系技术支持
